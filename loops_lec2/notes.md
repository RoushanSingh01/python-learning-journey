# while loop — use when waiting for a condition to change, pair with True/False
# for loop — use when you know what you're looping over (list, dict, range)
# range — use when looping by count, list alone can't scale for that
# list — ordered collection, indexed by numbers
# dict — indexed by words/keys, use when data has labels (name, house, etc)
# break — exits the loop immediately when condition is met
# len() — tells how many items are in a list, use with range() to loop safely
# None — absence of value, not zero, not empty string, literally nothing
